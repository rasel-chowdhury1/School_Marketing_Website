## About

This project is a marketing and discovery platform for venues in Mauritius. Users can explore venues by categories, book instantly, leave reviews, and scan QR codes for quick access. Venue creators can manage their listings, monitor performance, and run marketing campaigns through a dashboard.

---

## Features

### For Explorers (Users)
- Browse venues by categories: Hospitality, Wellness, Nightlife, etc.
- Book and review venues directly.
- Map exploration to find nearby spots.
- Scan venue QR codes for instant access.

### For Venue Creators (Businesses)
- Access to a powerful dashboard with user statistics & demographics.
- Send targeted notifications to users.
- Run advanced marketing campaigns.
- Receive monthly performance reports.

### General
- Fully responsive design for mobile and desktop.
- Dark and light themed sections for UI clarity.
- SEO optimized with meta tags, Open Graph, and Twitter cards.
